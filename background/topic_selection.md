## 三、候选分析（Top 3）

### 候选 1：#163 预测停车违规 ⭐ 最终选定

**提供方**: Egis（国际工程咨询集团，19000+员工，100+国家运营）  
**难度**: Basic | **数据**: 10MB-1GB（真实数据） | **指标**: Spearman 相关系数

#### 问题描述
Egis 运营多个法国/国际城市的路边停车执法系统。配备摄像头的执法车辆拍摄违规停车照片。**核心目标：预测各区域各时间段的违规率，优化执法车辆路线规划。**

#### 数据特征
| 特征 | 说明 |
|------|------|
| 经纬度 | 地理区域坐标 |
| 时间戳 | 日期+时间，按小时采样 |
| 天气数据 | 来自 Météo France 开放数据 |
| **目标变量** | 违规率（当前到下一时间戳之间） |

#### 基线
- Random Forest，10 棵树
- 仅使用检查了至少 45 辆停放车辆的数据
- **基准分数: 0.197**（测试集 Spearman 相关系数）— 几乎接近随机水平

#### 选择理由

| 维度 | 评价 |
|------|------|
| **保底能力** | ★★★★★ — Basic 难度 + 真实数据 + 标准回归问题 |
| **冲高分潜力** | ★★★★★ — 基线仅 0.197，极易大幅超越 |
| **精力投入** | ★★★★☆ — 适中，GPS+天气需要一些特征工程 |
| **展示效果** | ★★★★★ — 地图可视化 + 城市智慧交通故事 |

**核心优势：**
1. ✅ 真实数据，完全满足课程 "real-world dataset" 要求
2. ✅ 基线极低 (0.197)，报告中 "比 baseline 提升 X 倍" 非常抢眼 → Technical Elements 高分
3. ✅ GPS 坐标 + 时间 + 天气 → 特征工程素材丰富 → Preprocessing 环节充实
4. ✅ 可用 folium/plotly 做交互式地图热力图 → Presentation 加分
5. ✅ 不需要专业领域知识，降低学习成本

**预估分数**: 保底 12-13/15，冲高 14-15/15

### 候选 2：#159 帕金森病运动评分（备选，有风险）

**提供方**: Institut du Cerveau（巴黎大脑研究所）  
**难度**: Basic | **数据**: <10MB | **指标**: RMSE

#### 问题描述
预测并校正帕金森病患者 OFF 状态运动评分偏差。数据包含 ~10,000 名患者、~100,000 条记录。

#### ⚠️ 关键风险
数据是**合成的**（"Synthetic data modeled after real Parkinson's cohorts"），可能不满足课程 "real-world dataset" 要求，在 Correctness 评分上有被扣分的风险。

#### 优势
- 数据极小（<10MB），迭代快
- 医学故事感人，展示效果好
- RMSE 是最标准的指标

**预估分数**: 保底 11-12/15，冲高 13-14/15

---

### 候选 3：#161 保险保费预测（适合能力较强团队）

**提供方**: Crédit Agricole Assurances（欧洲最大银保机构）  
**难度**: Intermediary | **数据**: 10MB-1GB | **指标**: RMSE

#### 问题描述
预测多风险农业保险的纯火灾保费。需要建立两个子模型：
- Frequency Model（索赔频率，Poisson 分布）
- Average Cost Model（平均成本，Tweedie 分布）
- 最终目标：`CHARGE = FREQ × CM × ANNEE_ASSURANCE`

#### 优势
- 真实商业数据，Intermediary 难度技术深度分更高
- 两阶段建模方法论丰富

#### 风险
- 匿名化特征限制了特征理解
- 需要保险精算知识
- 精力投入明显更大

**预估分数**: 保底 11-12/15，冲高 14-15/15

## 五、结论

**最终选定：#163 预测停车违规 (Predict Parking Violations!)**
