# CS5483 Project 2 — Predict Parking Violations

> ChallengeData #163 "Predict Parking Violations" by Egis

## Overview

Predict parking violation rates (`invalid_ratio`, 0–1) across urban zones in Thessaloniki, Greece, using GPS coordinates, weather data, and temporal features.

- **Data source**: [ChallengeData #163](https://challengedata.ens.fr/challenges/163)
- **Metric**: Spearman correlation coefficient
- **Official baseline**: 0.197 (Random Forest, 10 trees)
- **Current best**: 0.5222 (LightGBM + XGBoost ensemble, submitted to platform)

## Quick Start

### 1. Clone the repository

```bash
git clone https://github.com/CUBWB7/CS5483-Predict-parking-violations.git
cd CS5483-Predict-parking-violations
```

### 2. Set up the environment

```bash
# Create conda environment
conda env create -f environment.yml
conda activate parking

# Register as Jupyter kernel (for JupyterLab / Group Server)
python -m ipykernel install --user --name parking --display-name "Python (parking)"

# Verify installation
python -c "import lightgbm, xgboost, catboost, shap; print('All packages OK')"
```

**Key dependencies**: Python 3.11, LightGBM, XGBoost, CatBoost, scikit-learn, SHAP, pandas, matplotlib, seaborn

### 3. Download the data

The raw CSV files (~500 MB total) are too large for GitHub. Download from the competition page and place them in the `163-Predict parking violations/` directory:

1. Visit https://challengedata.ens.fr/challenges/163
2. Download and place:
   - `x_train_final_asAbTs5.csv` — Training features (~6.07M rows)
   - `y_train_final_YYyFil7.csv` — Training target
   - `x_test_final_fIrnA7Q.csv` — Test features (~2.03M rows)

### 4. Run the notebooks in order

| # | Notebook | Purpose | Output |
|---|----------|---------|--------|
| 1 | `notebooks/01_eda.ipynb` | Exploratory data analysis | 7 figures in `docs/figures/` |
| 2 | `notebooks/02_feature_engineering.ipynb` | Feature engineering (10 → 26 features) | `data/*.parquet` files |
| 3 | `notebooks/03_modeling.ipynb` | LightGBM + XGBoost + Ensemble | `submissions/*.csv` files |

**Important**: Run them sequentially — each notebook depends on the previous one's output. Use **Restart & Run All** to ensure reproducibility.
