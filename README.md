# CS5483 Project 2 — Predict Parking Violations

> ChallengeData #163 "Predict Parking Violations" by Egis

## Overview

Predict parking violation rates (`invalid_ratio`, 0–1) across urban zones in Thessaloniki, Greece, using GPS coordinates, weather data, and temporal features. The dataset comes from THESi roadside parking system with ~6.07M training rows and 10 raw features.

- **Data source**: [ChallengeData #163](https://challengedata.ens.fr/challenges/163)
- **Metric**: Spearman correlation coefficient
- **Official baseline**: 0.197 (Random Forest, 10 trees)
- **Current best**: **0.5705** (rank-target LGB+XGB Ensemble I-A, Platform score, ranked #5 — best achieved)

## Current Results

### Version History

| Version | Key Change | OOF Spearman | Platform Score | Notes |
|---------|-----------|-------------|----------------|-------|
| v1 | Baseline LGB + XGB | 0.5880 | 0.5222 | 26 features, 3000 rounds |
| v2 | Lower LR + more rounds | 0.6012 | 0.5338 | 8000 rounds, LR 0.03 |
| v3 | Optuna hyperparameter tuning | 0.6408 | 0.5620 | 50 trials per model |
| v4 | + CatBoost Optuna (GPU) | 0.6408 | — | CB weight=0, no ensemble gain |
| v5 | + Grid×Month TE (Tier 3) | 0.6408 | — | Marginal, feature space saturated |
| v6 | Spearman ES + feature pruning | 0.6377 | 0.5618 | ES not viable for Spearman |
| **v7** | **+ log1p(tc) sample weighting** | **0.6429** | **0.5636** | Former best |
| **Exp C** | **Rank-target LGB+XGB** | **0.6464** | **0.5698** | Sprint best |
| **Exp I-A** | **LGB n_iter 10K→20K** | **0.6478** | **0.5705** | **Current best, #5** |

### Negative Results (Steps 11–14)

| Experiment | Method | Result | Takeaway |
|-----------|--------|--------|----------|
| v8a | M1-5 temporal TE (test only) | Platform 0.5507 | Full-data TE is actually better |
| v9/v9a | Strong regularization (Optuna) | OOF -0.010 | Gap is NOT caused by overfitting |
| v10 | DART boosting | OOF -0.019 | Dropout hurts on 6M noisy data |
| v11 | Neural Network (MLP/ResNet) | NN OOF 0.42 | NN too weak for ensemble contribution |

### Key Technical Findings

- **Strongest features**: `grid_period_te` (ρ=0.311), `grid_te` (ρ=0.307), `total_count` (ρ=-0.295)
- **OOF-Platform gap**: ~0.079, caused by TE distribution shift (not overfitting)
- **Test set**: Contains only months 1–5 (discovered via analysis)
- **total_count=1 noise**: 25% of samples have binary violation rates (Spearman only 0.41)
- **LGB-XGB correlation**: 0.965 — limited ensemble diversity
